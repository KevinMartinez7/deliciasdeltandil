# Imágenes de Servicios

Coloca aquí las imágenes de fondo para el carousel de servicios.

## Nombres de archivos esperados:

1. **bodas.jpg** - Imagen de boda elegante
2. **corporativo.jpg** - Evento corporativo profesional
3. **fiestas.jpg** - Fiesta privada
4. **buffet.jpg** - Buffet y cocktails
5. **domicilio.jpg** - Catering a domicilio
6. **tematicos.jpg** - Menú temático

## Especificaciones:

- Formato: JPG o PNG
- Resolución recomendada: 1920x1080 píxeles (Full HD)
- Peso: optimizado para web (menos de 800KB por imagen)
- Calidad: Alta, pero comprimida para web
- Orientación: Horizontal (landscape)

Mientras no agregues las imágenes, se mostrarán gradientes de colores de la marca.
