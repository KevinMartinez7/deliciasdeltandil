# Imágenes de Eventos

Coloca aquí las imágenes de eventos de Delicias del Tandil que se mostrarán en el carousel.

## Nombres de archivos esperados:

1. **boda-1.jpg** - Evento de boda elegante
2. **corporativo-1.jpg** - Evento corporativo
3. **fiesta-1.jpg** - Fiesta premium
4. **boda-2.jpg** - Ceremonia al aire libre
5. **corporativo-2.jpg** - Conferencia empresarial
6. **fiesta-2.jpg** - Cumpleaños especial

## Especificaciones:

- Formato: JPG o PNG
- Resolución recomendada: 1200x800 píxeles (aspect ratio 3:2)
- Peso: optimizado para web (menos de 500KB por imagen)
- Calidad: Alta, pero comprimida para web

Mientras no agregues las imágenes, se mostrarán placeholders con gradientes de colores.
